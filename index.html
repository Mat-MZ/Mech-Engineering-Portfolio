    Differential Assembly

    This project entailed creating a drivetrain system that would fit into the required spacing of a Formula Electric F1 car for the Formula Electric SAE team at Cal Poly Pomona. The system features a chain guard which wraps around the chain and sprocket mechanism of the drivetrain. The Assembly also features the differential assembly highlighted in gold just behind the sprockets. The drivetrain system also features an Emrax 228 axial flux motor situated in the back of the assembly highlighted in grey. The entire system is connected to the main frame of the Formula Electric F1 car through the mounting system in black, which utilizes mounting bars. This project was accomplished by the Drivetrain team and myself.

    RC Car Assembly

    This project focused on modeling the internal components of an RC Car. The massive undertaking consisted of over 80 individual components which were meticulously mated into one final master assembly. The final assembly featured several moving components such as the axle, motor, and suspension. There were several parts that contained unique geometry that posed a challange to model. Each part was measured individually using a caliper. The project involved four people including myself. Communication between the four of us was important in order to ensure correct dimensions for the final assembly. This project was also done on a time crunch, at the height of finals season and only two weeks to complete.

    Manufactured Truck

    The main body of this truck was created through the use of green-sand casting. The features on the truck such as the headlights and windows were etched into the frame using CNC milling. The bed of the truck was made using metal sheet bending and cutting machines as well as rivets to hold the bed together. Each bolt hole was first milled using a bridgeport machine and then hand tapped to create the threads for each bolt used. The wheels were machined from a long cylinder of aluminum. The stand in which the truck rests was created by MIG welding several pieces of metal together and is an original design.
